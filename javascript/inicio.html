Guía Básica de JavaScript

¿Qué es JavaScript?

JavaScript es un lenguaje de programación interpretado que se ejecuta principalmente en los navegadores web. Sus principales usos incluyen dar interactividad a las páginas, manipular el contenido de un documento HTML, responder a eventos del usuario y comunicarse con servidores para actualizar datos sin recargar la página.

Fue creado en el año 1995 por Brendan Eich en Netscape, con el propósito de hacer las páginas web más dinámicas y fáciles de usar.

Aunque sus nombres se parecen, JavaScript y Java no son lo mismo. Java es un lenguaje de programación orientado a objetos que se ejecuta en una máquina virtual, mientras que JavaScript fue diseñado como un lenguaje ligero y flexible para ejecutarse en navegadores.

JavaScript se complementa con HTML y CSS. HTML define la estructura del contenido, CSS define su estilo visual y JavaScript aporta la lógica e interactividad.

Fundamentos del lenguaje

La sintaxis básica de JavaScript utiliza instrucciones terminadas en punto y coma, aunque este no siempre es obligatorio. Los comentarios se escriben con // para una sola línea o con /* ... */ para varias líneas.

Existen tres formas de declarar variables: var (forma antigua, con alcance global o de función), let (alcance de bloque y más segura) y const (para valores que no cambian).

Los tipos de datos primitivos en JavaScript son: string (cadenas de texto, por ejemplo "Hola"), number (números como 25 o 3.14), boolean (valores lógicos true o false), null (ausencia intencional de valor), undefined (valor no asignado) y symbol (identificadores únicos).

Un array es una colección ordenada de elementos a la que se accede por índices, mientras que un objeto es una colección de pares clave-valor que describe características de algo.

Los operadores aritméticos (+, -, *, /, %) sirven para cálculos matemáticos. Los operadores lógicos (&&, ||, !) permiten evaluar condiciones compuestas. Los operadores relacionales (>, <, >=, <=, ==, ===) comparan valores.

Control de flujo

Las sentencias condicionales if y else permiten ejecutar código dependiendo de si una condición se cumple o no. switch evalúa una expresión y ejecuta el bloque de código que coincida con el caso correspondiente.

El bucle for se usa cuando se conoce la cantidad de repeticiones, while se ejecuta mientras la condición sea verdadera y do...while asegura al menos una ejecución antes de verificar la condición.

La palabra clave break interrumpe un bucle por completo, mientras que continue salta a la siguiente iteración sin terminar el ciclo.

Funciones

Una función se declara con la palabra function seguida de un nombre y paréntesis. Se llama escribiendo su nombre y los paréntesis. Por ejemplo: function saludar() { alert("Hola"); } saludar();

Las funciones pueden tener parámetros, que son valores de entrada, y un valor de retorno, que es el resultado que entregan al ser ejecutadas.

Las funciones flecha (=>) son una forma más corta de escribir funciones, no crean su propio this y suelen ser más fáciles de leer en código moderno.

El scope o alcance determina dónde una variable es accesible. Puede ser global, de función o de bloque según cómo se haya declarado.

Objetos y colecciones

Un objeto se declara con llaves y pares clave-valor. Ejemplo: let persona = { nombre: "Ana", edad: 25 };. Se accede con notación de punto: persona.nombre.

Una propiedad es un valor asociado a una clave, mientras que un método es una función dentro del objeto.

Algunos métodos básicos de los arrays son push (agrega un elemento al final), pop (elimina el último elemento) y map (crea un nuevo array aplicando una función a cada elemento).

Otros temas básicos

El DOM (Document Object Model) es la representación estructurada de una página web que permite a JavaScript acceder y modificar su contenido dinámicamente.

Un evento es una acción que ocurre en la página, como un clic del usuario, presionar una tecla o mover el ratón. JavaScript puede responder a estos eventos para crear interactividad.

alert muestra un mensaje en una ventana emergente, prompt solicita al usuario que ingrese un valor y console.log imprime mensajes en la consola del navegador.

La consola del navegador se abre generalmente con F12 o Ctrl+Shift+I. Es muy útil para depurar código porque permite ver errores, probar comandos y verificar valores de variables en tiempo real.
